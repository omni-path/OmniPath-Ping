# Figure 15

Run from the repository root:

```bash
python3 experiment_reproduction/figure15/run_figure15.py
```

The runner executes Figure 15.

Results are written to:

```text
experiment_reproduction/figure15/results/
```

Default output files:

```text
experiment_reproduction/figure15/results/Figure15(a)_results.csv
experiment_reproduction/figure15/results/Figure15(b)_results.csv
```
