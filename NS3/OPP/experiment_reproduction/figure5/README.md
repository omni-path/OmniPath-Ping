# Figure 5

Run from the repository root:

```bash
python3 experiment_reproduction/figure5/run_figure5.py
```

The runner executes Figure 5 serially.

Results are written to:

```text
experiment_reproduction/figure5/results/
```

Default output files:

```text
Figure5(a)_results.csv
Figure5(b)_results.csv
Figure5(c)_results.csv
Figure5(d)_results.csv
```
